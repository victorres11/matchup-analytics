How to Use in Your App

In your main application, you can:

  • Listen for the matchupRouteChange event
  • Use window.matchupRouter.getCurrentMatchup() to get current route info
  • Use window.matchupRouter.navigateToMatchup(team, week) to navigate programmatically

Example JavaScript:

// Listen for route changes
window.addEventListener('matchupRouteChange', function(event) {
    const { team, week, route } = event.detail;
    console.log(`Navigated to: ${team} Week ${week}`);
    // Update your app UI here
});

// Get current matchup
const current = window.matchupRouter.getCurrentMatchup();
if (current) {
    console.log(`Current: ${current.team} Week ${current.week}`);
}
